# 第13讲：新家入住前的风水检查

**模块五：实战综合应用（上）**


## 入住前必做的五件事

**第一件：彻底清洁**

新家入住前，必须彻底清洁一遍。

不只是表面的清洁，是每一个角落，包括：

- 所有柜子的内部
- 地板的每一个角落
- 窗户和窗台
- 卫生间和厨房的每一个细节

清洁，是清除旧气场，为新气场的建立做准备。

如果是二手房，清洁尤其重要——你需要清除前任住户留下的气场痕迹。

**第二件：通风三天**

入住前，把所有窗户打开，通风至少三天。

通风，是让旧气散出，新气进入。

装修完的新房，有大量的甲醛和装修气味，通风也是健康的必要步骤。

**第三件：检查并修复所有损坏**

入住前，检查所有的门、窗、水管、电路。

有损坏的，入住前修复。

破损的门窗，是气场的"漏洞"，气会从这里散失。

漏水的水管，是财运的"漏洞"，财气会从这里流失。

**第四件：确认家具摆放**

根据前面几讲学到的原则，确认主要家具的摆放：

- 床头靠实墙，不靠窗，不对门
- 沙发背靠实墙
- 书桌背靠实墙，面向开阔方向
- 灶台不对门

入住前调整好，比入住后再调整容易得多。

**第五件：选择入住时间**

入住时间，选择阳气旺盛的时段：

- 时间：上午9点到下午3点之间（阳气最旺）
- 日期：避开农历初一、十五（气场转换节点）
- 避开：雨天、大风天（气场不稳定）
