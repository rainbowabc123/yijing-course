# 第08讲：卧室——休养生息的空间

**模块三：家居各空间的风水（三）**


## 卧室的核心功能：藏气、养气

藏气，是让气在卧室里聚集，不散失。

养气，是让气在睡眠中滋养你的身体和精神。

要实现藏气和养气，卧室需要：

- 相对封闭，不要有太多开口
- 安静，减少外部干扰
- 光线可以调节，白天可以遮光
- 整洁，气流顺畅


## 卧室颜色与阴阳平衡

卧室是休息的空间，需要阴气足，安静稳定。

**适合卧室的颜色：**

- 米白、浅灰、浅蓝：安静、稳定，有助于放松
- 浅绿：木属性，生机，但不要太鲜艳
- 浅粉、浅紫：柔和，适合卧室

**不适合卧室的颜色：**

- 大红、橙色：火属性，过于热烈，影响睡眠
- 深黑：阴气过重，容易压抑
- 鲜艳的黄色：过于刺激，影响放松


## 本讲小结

卧室的核心功能是藏气、养气。

**床位的四个禁忌：压梁、对门、对镜、靠窗。**

**卧室颜色选择柔和、安静的色调，避免过于鲜艳或深暗。**
