# 第03讲："上吉之宅"的结构特征

**模块二：选宅与选房（上）**


## 什么是"上吉之宅"

《易经》坤卦说：**"地势坤，君子以厚德载物。"**

好的居住环境，像大地一样：厚实、稳定、有承载力。

"上吉之宅"，有三个结构条件：

**第一：藏风聚气**

气，遇风则散。

好的住宅，能够藏住风，让气聚集而不散。

具体表现：

- 不是风口位置（不在两栋楼之间的穿堂风通道）
- 有一定的遮挡，不是完全暴露在外
- 小区内部有绿化，能减缓风速

**第二：明堂开阔**

明堂，是房子前方的开阔空间。

好的住宅，前方视野开阔，没有遮挡。

具体表现：

- 窗外视野开阔，不是正对着另一栋楼的墙
- 楼层不要太低，不被周围建筑完全遮挡
- 采光充足，白天不需要开灯

**第三：背有靠山**

靠山，是房子后方的支撑。

好的住宅，背后有依靠，不是悬空的。

具体表现：

- 背后有山、有高楼、有实墙，不是悬崖或空旷地带
- 小区背后有绿化或建筑，不是直接面对空旷的道路
- 卧室床头靠实墙，不是靠窗


## 选房自查清单

看房时，用这个清单快速判断：

**必须满足（基础条件）：**
- [ ] 采光充足，白天不需要开灯
- [ ] 通风良好，不是密闭空间
- [ ] 格局方正，没有大量奇形怪状的角落
- [ ] 动线顺畅，从门口到各个房间不需要绕行

**加分项（结构优势）：**
- [ ] 客厅窗外视野开阔
- [ ] 不在穿堂风位置
- [ ] 背后有依靠（山、高楼、绿化）
- [ ] 小区环境整洁，绿化好

**减分项（需要注意）：**
- [ ] 正对电梯口或楼梯口
- [ ] 窗外正对另一栋楼的墙
- [ ] 格局不方正，有大量斜角
- [ ] 采光差，长期需要开灯
