# 第01讲：风水不是玄学，是环境结构学

**模块一：易经风水的底层逻辑（上）**


## 风水的本质：气的流动与聚散

"风水"这个词，来自晋代郭璞的《葬经》：

**"气乘风则散，界水则止。古人聚之使不散，行之使有止，故谓之风水。"**

气，遇到风就散开，遇到水就停止。

古人让气聚集而不散，让气流动而有所止，这就是风水。

风水的核心，是**气的流动与聚散**。

气，不是神秘的东西。

在现代语言里，气可以理解为：空间里的能量流动，包括光线、空气、动线、声音、温度。

一个空间里，这些要素流动顺畅，气就好。

这些要素堵塞、混乱、失衡，气就差。

**风水，是环境结构学，不是玄学。**


## 破除误区：哪些"风水说法"是伪科学

**误区一：摆件能改变风水**

摆件，是形的一部分，但只是很小的一部分。

一个空间的气场，主要由格局、光线、整洁度决定。

一个摆件，改变不了整体的气场结构。

**误区二：风水是固定的，算一次就够了**

风水有时间维度，是动态的。

你的运势在变，外部环境在变，家里的气场也在变。

需要定期观察和调整，不是算一次就永久有效。

**误区三：风水只和房子有关**

《易经》的终极风水观是：**人是最大的风水。**

你的状态、你的能量场，比房子的风水更重要。

一个状态好的人，住在普通的房子里，也能把气场带起来。

一个状态差的人，住在再好的风水宝地，也会把气场带坏。
