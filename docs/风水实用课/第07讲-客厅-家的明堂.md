# 第07讲：客厅——家的"明堂"

**模块三：家居各空间的风水（二）**


## 沙发背靠实墙的易经逻辑

客厅最重要的家具，是沙发。

沙发的摆放，有一个核心原则：**背靠实墙。**

为什么？

《易经》艮卦，山，象征稳定、依靠、支撑。

背后有靠，是稳定的结构。

背后无靠，是悬空的结构，气场不稳。

沙发背靠实墙，人坐在上面，背后有依靠，心理上感到安全，气场稳定。

沙发背对窗户或走道，背后无靠，气流从背后穿过，人容易感到不安、疲惫。

**如果客厅格局限制，沙发无法靠实墙，可以在沙发背后放一个矮柜或屏风，制造"靠山"。**


## 家里这三个地方乱，财运就很难进来

结合上一讲的玄关，加上客厅，再加上厨房，这三个地方是财运的"承载位"。

**玄关乱：气口堵塞，财运进不来。**

**客厅乱：明堂受阻，财运聚不住。**

**厨房乱：财位受损，财运留不住。**

三个地方，今天就能整理，不需要花一分钱。


## 本讲小结

客厅是家的明堂，是气聚集的核心区域。

**沙发背靠实墙，是客厅最重要的风水原则。**

**三个最影响运势的细节：整洁度、光线、绿植。**
