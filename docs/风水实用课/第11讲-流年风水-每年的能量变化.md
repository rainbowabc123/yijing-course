# 第11讲：流年风水——每年的能量变化

**模块四：时间风水——运势的节律（中）**


## 流年九宫飞星的基本逻辑

九宫飞星，是流年风水的核心工具。

九宫，是把家居空间分成九个区域：中、东、西、南、北、东南、东北、西南、西北。

飞星，是九个数字（1-9），每年按照固定的规律，在九宫之间"飞动"。

每个数字，代表不同的能量属性：

| 星数 | 属性 | 影响 |
|------|------|------|
| 1白 | 水星 | 智慧、事业、贵人 |
| 2黑 | 土星 | 疾病、是非 |
| 3碧 | 木星 | 争执、官非 |
| 4绿 | 木星 | 文昌、学业、桃花 |
| 5黄 | 土星 | 最凶，五黄煞 |
| 6白 | 金星 | 权威、偏财 |
| 7赤 | 金星 | 口舌、破财 |
| 8白 | 土星 | 财运、旺气 |
| 9紫 | 火星 | 喜庆、名声 |


## 流年风水的简单应对方法

**旺位的利用：**

西南方（8白财运旺位）：可以在此方位放绿植、灯光，激活财运。

东方（4绿文昌位）：书桌放在东方，有助于学业和工作。

西北方（1白贵人位）：工作区放在西北，有助于贵人相助。

**凶位的化解：**

东南方（5黄煞）：2026年最需要注意的方位。

- 避免在东南方动土、装修
- 不要在东南方放红色物品（火生土，加重五黄）
- 可以放金属物品（金克土，化解五黄）
- 避免在东南方长期停留

北方（2黑疾病位）：

- 注意北方区域的卫生和整洁
- 避免在北方长期停留
- 可以放金属物品化解


## 本讲小结

流年风水，是每年根据能量变化调整家居布局的智慧。

**2026年重点：东南方五黄煞需要注意，西南方财运旺位可以激活。**
