# 第05讲：朝向与方位的易经逻辑

**模块二：选宅与选房（下）**


## 八卦方位与家居功能区

《易经》八卦，对应八个方位：

| 卦象 | 方位 | 属性 | 家居对应 |
|------|------|------|----------|
| 乾 | 西北 | 天、父、领导 | 主卧、书房 |
| 坤 | 西南 | 地、母、承载 | 主卧、餐厅 |
| 震 | 东 | 雷、长子、行动 | 书房、工作区 |
| 巽 | 东南 | 风、长女、财富 | 财位、植物区 |
| 坎 | 北 | 水、中男、智慧 | 书房、学习区 |
| 离 | 南 | 火、中女、名声 | 客厅、明亮区 |
| 艮 | 东北 | 山、少男、积累 | 储物区、书房 |
| 兑 | 西 | 泽、少女、表达 | 客厅、社交区 |

这个对应关系，是一个参考框架，不是绝对的规则。

核心逻辑是：让空间的功能，与方位的能量属性相匹配。


## 书桌、灶台的方位选择逻辑

**书桌方位：**

书桌，是工作和学习的空间，需要稳定、专注的能量。

最佳方位：东方（震卦，行动力强）或北方（坎卦，智慧、专注）。

书桌摆放原则：

- 背后靠实墙，不要背对门口
- 面向开阔方向，不要面对墙壁
- 避免正对窗户（光线直射，容易分心）

**灶台方位：**

灶台，对应离卦，火，是家里的财位核心。

灶台方位的核心原则：

- 灶台不要正对门口（气直冲灶台，财气散）
- 灶台不要背对窗户（气从背后散）
- 灶台周围保持整洁，不堆放杂物


## 本讲小结

方位，对居住者的状态有真实影响。

**床头最佳朝向：南或东，必须靠实墙。**

**书桌最佳方位：东或北，背靠实墙，面向开阔。**

**灶台核心原则：不对门，不背窗，保持整洁。**
