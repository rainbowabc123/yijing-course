【模块一：判断力基础】第03讲 | "既济"时刻：一切看起来都很好


易经六十四卦，第六十三卦是既济。

既，是已经。济，是渡过。

既济，就是已经渡过了，事情已经完成了，一切都到位了。

听起来是好事。

但易经把既济放在倒数第二卦，紧接着是未济——尚未完成。

这个排列，不是偶然的。

它在告诉你：一切看起来都很好的时候，才是最危险的时刻。


为什么既济最危险？

因为既济会让你的判断失真。

当一切顺利的时候，人会产生一种错觉：这种状态会持续下去。

会觉得自己的判断是对的，自己的方向是对的，自己的位置是稳固的。

但易经说：既济之后，就是未济。

顺的尽头，是逆的开始。

不是说顺境一定会变成逆境，而是说：在顺境里，你最容易忽略那些正在悄悄积累的风险。


既济时刻的三个误判

第一个误判：以为现在的状态会持续。

事业顺了，以为会一直顺。

关系好了，以为会一直好。

钱来了，以为会一直来。

但结构是会变的。

你现在的顺，是因为当前的结构对你有利。

结构一旦变化，顺就会变成逆。

第二个误判：以为自己的判断是对的。

顺境会给你一种反馈：你做的是对的。

但这个反馈，有时候是假的。

你做对了，所以顺。

但也可能是：结构本来就在顺，你做什么都会顺。

一旦结构变了，你才会发现，你的判断其实一直有问题，只是被顺境掩盖了。

第三个误判：以为可以加速。

顺的时候，人会想：既然这么顺，那就多做一点，快一点，大一点。

但"亢龙有悔"说的就是这件事。

龙飞到最高处，再往上，就是悔。

顺境里的过度扩张，是很多人从既济走向未济的直接原因。


如何在既济时刻保持清醒？

记住一句话：顺的时候，检查结构。

不是说要悲观，不是说要停下来，而是说：在一切都好的时候，主动去看那些你平时不愿意看的东西。

你的关系，有没有正在被忽视的裂缝？

你的事业，有没有正在积累的风险？

你的位置，有没有正在松动的基础？

既济时刻，是最好的检查时机。

因为你还有余力，还有资源，还有时间。

等到未济来临，再检查，就晚了。


本讲小结

既济，是一切已经到位的状态，但易经把它放在倒数第二卦，紧接着是未济。

既济最危险，因为它会让判断失真：以为状态会持续、以为判断是对的、以为可以加速。

顺境里的过度扩张，是从既济走向未济的常见路径。

既济时刻的正确做法：检查结构，主动看那些被顺境掩盖的风险。

自查清单

1. 你现在的某个处境，是否处于"既济"状态——一切看起来都很好？
2. 在这个顺境里，你有没有正在忽视的某个裂缝或风险？
3. 你有没有在顺境里做过"加速扩张"的决定，后来发现代价很高？
4. 如果你现在的顺境突然结束，你的哪个位置最脆弱？
5. 你上一次主动检查自己的结构，是什么时候？
