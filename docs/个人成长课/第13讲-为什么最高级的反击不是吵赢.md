# 第13讲：为什么《易经》说，最高级的反击不是吵赢

**模块五：与人相处的智慧（上）**


## 艮卦：止，是一种力量

艮卦，是《易经》第五十二卦，卦象是艮为山。

上下都是艮，山。

艮卦的卦辞说：**"艮其背，不获其身，行其庭，不见其人，无咎。"**

止于背，不执着于自身；走过庭院，不在意他人，没有过失。

艮，是止，是停止，是不动。

但这里的止，不是软弱，不是退缩，是一种主动的选择。

**止，是不让对方的情绪和行为，进入你的系统。**

当有人攻击你、误解你、挑衅你，你的第一反应是什么？

大多数人的反应是：反击。

但反击，意味着你接受了对方设定的战场，按照对方的规则在打。

止，是拒绝进入这个战场。


## 留余地的易经逻辑

《易经》有一个反复出现的原则：**无咎。**

无咎，不是大吉，是没有过失，是不留后患。

留余地，是无咎的智慧。

你把话说死了，把关系搞僵了，把对方逼到墙角了，你可能赢了当下，但你留下了后患。

后患，会在未来某个时候，以你意想不到的方式回来。

**留余地，不是软弱，是在保护未来的可能性。**

具体怎么做？

**第一，不要把话说满。**

"你永远都是这样"，"你从来不……"，这类话，把对方逼死了，也把自己逼死了。

留一点空间，让对方有台阶下，关系才能继续。

**第二，不要在情绪激动时做决定。**

艮卦说止，就是在情绪激动时，先停下来。

等情绪平复了，再做判断，再说话。

**第三，不要把每一次冲突都当成必须赢的战役。**

有些冲突，让步是智慧，不是失败。

你让步，不是因为你错了，是因为这件事不值得你全力以赴。


## 本讲小结

最高级的反击，不是吵赢，是止。

**止，是不进入对方设定的战场。**

**留余地，是保护未来的可能性。**

**用行动证明，比用语言争论，更有力量。**
