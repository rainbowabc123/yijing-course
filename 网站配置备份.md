## 四、关键配置文件

### 4.1 mkdocs.yml
```yaml
site_name: 易经课程学习中心
site_description: 用结构视角，看清处境与位置
site_author: 未济之思

theme:
  name: material
  language: zh
  palette:
    scheme: default
  features:
    - navigation.tabs
    - navigation.sections
    - navigation.expand
    - navigation.top
    - search.suggest
    - search.highlight

extra_css:
  - stylesheets/custom.css

plugins:
  - search:
      lang: zh
  - encryptcontent:
      title_prefix: '🔒 '
      summary: '此内容已加密，请输入课程密码后查看。'
      placeholder: '请输入密码'
      decryption_failure_message: '密码错误，请重试。'
      encryption_info_message: '本课程内容受密码保护。'
      password_button: True
      password_button_text: '确认'
      remember_password: True
      session_storage: True

markdown_extensions:
  - toc:
      permalink: true
  - admonition
  - pymdownx.details
  - pymdownx.superfences
```

### 5.2 首页特殊样式
- 首页侧边栏隐藏：`body:has(.home-page) .md-sidebar--primary { display: none !important; }`
- 主内容区域居中：`max-width: 960px; margin: 0 auto;`
- 课程卡片网格布局：`grid-template-columns: repeat(2, 1fr);`

### 5.3 移动端响应式
```css
/* 平板/手机横屏 (≤ 76.1875em) */
@media screen and (max-width: 76.1875em) {
  .md-content {
    margin-left: 0 !important;
    max-width: 100% !important;
  }
  .md-content__inner {
    margin: 0 !important;
    padding: 1rem !important;
  }
}

/* 手机竖屏 (≤ 44.9375em) */
@media screen and (max-width: 44.9375em) {
  .home-courses {
    grid-template-columns: 1fr !important;
  }
}
```

  1. 移除 `.md-sidebar--primary { display: none !important; }`
  2. 保留主题原生的 toggle 功能
  3. 代码已推送到 GitHub (提交 b911189)
  4. Cloudflare Pages 应该已自动部署

- **下一步调试方向**:
  1. 检查 Cloudflare Pages 部署日志，确认最新代码已部署
  2. 使用浏览器开发者工具检查移动端的 JavaScript 错误
  3. 检查 Material 主题的 drawer toggle 机制是否被其他 CSS 干扰
  4. 考虑使用 `transform` 而不是 `display` 来控制侧边栏显示
